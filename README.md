# Learning LlamaIndex TS

## Running the Project

1. Clone the repository

1. Install dependencies
    ```bash
    yarn install
    ```

1. Run Qdrant with Docker:
    ```bash
    docker-compose up
    ``` 

1. Copy the `.env-default` file to `.env` and fill in the required environment variables.

1. Start the application:
    ```bash
    yarn dev
    ```
